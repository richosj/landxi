회원가입

LAND-XI 사용을 위한 계정을 만들기 위해 회원가입을 진행한다.

  • 로그인 화면에서 사용자가 사용신청을 위해 ‘회원가입’ 버튼을 클릭하여 회원가입 페이지로 이동한다.
  • 회원 가입을 위해 약관 동의 페이지에서 약관 확인 후 정보 입력 페이지로 이동한다.
  • 사용신청을 위한 사용자 기본정보를 등록한다.기본 정보에 ‘기관 선택’을 통해 LX 사용자, 또는 기관(남원시) 사용자로 선택한다.
  • 회원가입 완료 페이지에서 가입이 완료되었음을 확인한다.단, 사용 신청 완료 후 LX 담당자의 가입 승인 완료 처리가 되어야 로그인이 가능하며, LX 담당자의 승인 완료 처리 시 로그인이 가능하다는 안내 메일이 발송된다.
